#  Sistema Gestión de Rutas Escolares
## **Fullstack | Proyecto Educativo | API REST + Interfaz Web**

Plataforma web integral que se desarrollara con **Angular**, **Vite** y **Supabase** para la **gestión logística de transporte escolar**. Permitiendo el seguimiento GPS en tiempo real, administración de choferes, trazado de rutas mediante servidores de mapas y control de horarios para garantizar seguridad y eficiencia..
